# uefn-verse-learning-jp
Learning Verse in UEFN from a Blueprint developer's perspective  | Blueprint経験者のUEFN Verse学習記録



## このリポジトリについて

UnrealEngine の Blueprint経験者が、UEFN の Verse 言語をゼロから学ぶ記録です。

- 学習中のサンプルコードを順次追加していきます
- 「Blueprint ではこう書いていた」という比較視点を大切にしています
- 学習の過程をそのまま記録しているため、初歩的なコードや試行錯誤も含まれます。

ライセンスはUEFNのライセンスに準拠します。
> ⚠️ This repository contains Verse code for learning purposes.  
> Use of Verse and UEFN is subject to the [Epic Games EULA](https://www.unrealengine.com/en-US/eula/unreal).


## 対象読者

- Blueprint の基本（変数・関数・イベント）を理解している方
- テキストコードに挑戦してみたい UE 開発者
- UEFN・Verse を日本語で学びたい方
